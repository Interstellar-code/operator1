# TOOLS.md — Trinity's Local Notes

Skills define _how_ tools work. This file is for _your_ specifics — environment details unique to this setup.

## What Goes Here

Things like:

- Bank and financial account references (no credentials!)
- Accounting tool details
- Invoice/billing platform info
- Cloud provider billing dashboard URLs
- API provider pricing pages bookmarked
- Spreadsheet/dashboard locations

## Examples

```markdown
### Financial Tools

- Accounting: Wave (wave.com dashboard)
- Invoicing: Stripe Billing
- Expense tracking: manual CSV in /workspace/expenses/

### Cloud Billing

- Fly.io: dashboard at fly.io/dashboard → billing
- Vercel: vercel.com/team/settings/billing
- GitHub: github.com/settings/billing

### API Providers

- OpenAI: platform.openai.com/usage
- Anthropic: console.anthropic.com/billing
- ZAI: internal (no separate billing dashboard)

### Docs

- Tax folder: ~/Documents/Tax/2026/
- Receipts: ~/Documents/Receipts/
```

## Why Separate?

Skills are shared across all agents. Your setup notes are yours. Keeping them apart means you can update skills without losing your notes.
